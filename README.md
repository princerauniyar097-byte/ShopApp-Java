# ShopApp Java Project

## Description
This project is a simple shop inventory system developed in Java.  
It allows users to view products, calculate total stock value, and buy stock.

## Technologies Used
- Java
- Console-based application

## How to Run
1. Open the project in DevC++ / VS Code
2. Compile all `.java` files
3. Run `ShopApp.java`
